Django Channels + Redis + PostgreSQL

BoardPulse

A real-time collaborative task board where task updates and presence sync instantly across connected users.

View Repository Read Documentation

What It Does

  • Authenticated WebSocket collaboration per board
  • Live task move broadcasts to all connected clients
  • Presence tracking with join and leave events
  • REST APIs for boards and tasks with auth protection

Architecture

  • ASGI with Django Channels consumers
  • Redis channel layer for group fan-out
  • PostgreSQL for persistent task and board state
  • Docker Compose orchestration with health checks

Deployment Note

GitHub Pages hosts this static project page only. The full live application needs a backend runtime for Django, Redis, and PostgreSQL. Use a cloud VM or a container host to run Docker Compose, then point users to that app URL.
